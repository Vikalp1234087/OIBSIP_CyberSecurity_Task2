# Task 2: Capture Network Traffic with Wireshark

## Objective
Capture and analyze network traffic using Wireshark to understand packets, protocols, and services.

## Tool Used
- Wireshark

## Steps Performed
1. Installed Wireshark and Npcap.
2. Started capturing traffic on the main network adapter.
3. Browsed websites to generate traffic.
4. Filtered HTTP traffic and analyzed packets.
5. Saved capture file `wireshark_capture.pcap`.
